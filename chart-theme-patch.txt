Instructions for updating charts with theme colors:

All chart creation functions should:
1. Call `const themeColors = getThemeColors();` before creating the chart
2. Use `color: themeColors.textColor` in legend.labels
3. Use `backgroundColor: themeColors.tooltipBg` in tooltip
4. Use `color: themeColors.textColor` in all ticks (x and y axes)
5. Use `color: themeColors.gridColor` in grid
